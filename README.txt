-DESCRIPTION-
User Authentication application

This following project is a website for a book store that allows clients to view and rent books, as well as view and update their account information.

-INSTALLATION-
In order to use the application the user will have to download the project from GitHub and then run the program using MAMP.
The user will also have to create the database in MAMP's PhpMyAdmin by importing the provided database file (found in the database folder).
